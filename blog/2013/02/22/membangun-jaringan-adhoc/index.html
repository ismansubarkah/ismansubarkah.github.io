Membangun Jaringan AdHoc pada Windows 8

Bagi anda, para pengguna laptop yang tidak terlalu sering membuat jaringan lokal secara nirkabel, mungkin postingan ini tidak terlalu bermanfaat. Namun, bagi para Gamer (pastinya yang menggunakan laptop) yang telah beralih sistem operasi dari Windows XP atau Windows 7 ke Windows 8, hal ini sangatlah bermanfaat, khususnya bagi para Gamer berbasis LAN contohnya Gamer Dotta. Fitur Ad-hoc atau AdHoc merupakan sebuah solusi bagi para Gamer yang ingin bermain game secara bersama-sama namun tidak adanya Hotspot atau sumber daya jaringan apapun yang memungkinkan mereka saling berinteraksi. Fitur AdHoc pada Windows 8 ini memang sedikit disembunyikan oleh pihak Microsoft sendiri. Hal ini tentunya dilakukan oleh Microsoft dengan alasan keamaan bagi pengguna Windows 8 itu sendiri. Akan tetapi kita masih bisa melakukan sedikit konfigurasi dengan mudah untuk mengaktifkannya.

Langkah-langkah Pengaktifan Jarian AdHoc

Berikut adalah langkah-langkah untuk mengaktifkan fitur ad-hoc:

  1. Buka start screen, atau tekan Windows + Q.

  2. Ketikan command atau cmd untuk lebih singkat.

  3. Pada sisi kiri, anda pasti akan melihat aplikasi Command Prompt.

  4. Klik kanan icon aplikasi tersebut, lalu akan muncul status bar di sisi bawah layar.

  5. Pilih mode aplikasi pada Run as Administrator.

  6. Kemudian akan muncul peringatan User Account Control, pilih Yes. Maka Command Prompt akan muncul.

  7. Lalu anda ketikan netsh wlan set hostednetwork mode=allow ssid=netadhoc key=12345678, langsung tekan Enter

  8. Kemudian akan muncul pesan bahwa Hosted Network sudah dalam keadaan allow

  9. Langkah terakhir, anda ketikan netsh wlan start hostednetwork. Yakinlah bahwa jaringan ad-hoc telah sukses diaktifkan. Mudah bukan :-).

Penjelasan Sintaks

Mungkin anda mulai bertanya-tanya mengenai apa maksud perintah yang tadi diberikan. Baiklah, saya akan menjelaskannya. netsh wlan set hostednetwork mode=allow ssid=netadhoc key=12345678 ialah perintah yang kita berikan untuk membuat jaringan ad-hoc dengan nama/SSID netadhoc dan passowrd=12345678. Jadi, jika anda ingin membuat jaringan ad-hoc dengan nama/SSID adalah kamargw dan passwordnya kamargw, anda tinggal ketikan netsh wlan set hostednetwork mode=allow ssid=kamargw key=kamargw. Untuk perintah netsh wlan start hostednetwork, perintah tersebut berarti kita mulai mengaktifkan jaringan ad-hoc tersebut. Jika anda ingin mematikan/menonaktifkan jaringan ad-hoc, anda tinggal ketikan netsh wlan stop hostednetwork. Selamat mencoba dan bermain game :-).
