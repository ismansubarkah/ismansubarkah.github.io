Windows Server 2003 akan dihentikan pada tanggal 14 Juli 2015 (1)


14 Juli 2015 merupakan hari bersejarah bagi Microsoft. Pada tanggal itu Windows Server 2003 tidak lagi didukung oleh Microsoft dalam artian Windows 2003 akan berakhir. Alasan mengapa Windows Server 2003 diberhentikan adalah karena Windows Server 2012 akan menjadi pengganti bagi Windows Server 2003 / R2. Dibawah ini adalah alasan dan dampak apabila pengguna tidak migrasi dari Windows Server 2003.



1. No Compliance

Sumber daya TI pada sebuah perusahaan secara rutin diaudit agar dapat selalu mendukung aktivitas bisnis. Salah satu subjek audit yang vital adalah IT/IS Security. Perusahaan yang masih menggunakan Windows Server 2003/R2 setelah tanggal 14 Juli tidak akan lolos audit. Perusahaan seperti visa dan mastercard tidak akan bersedia untuk bekerja sama dengan perusahaan yang masih menggunakan Windows Server 2003/R2 karena tidak compliance terhadap standari PCI.



2. No Safe Haven

Perusahaan yang menggunakan server virtual maupun fisik dan masih menggunakan Windows Server 2003/R2 dan Windows Small Business Server (SBS) 2003 setelah tanggal 14 juli 2015 akan rentan dari sisi keamanan dan tidak akan lolos audit keamanan TI/SI. Biaya operasional akan lebih tinggi untuk merawat hardware yang beranjak tua dan membeli perangkat-perangkat keamanan untuk melindungi server yang masih menggunakan window server 2003/R2.



3. No Updates

Pada tahun 2013, Microsoft hanya merilis 37 critical updates untuk Windows Server 2003/R2. Namun setelah tanggal 14 Juli 2015 microsoft tidak akan merilis update maupun patch keamanan untuk Windows Server 2003/R2. Sehingga perusahaan yang masih menggunakan windows server 2003 akan rentan menjadi sasaran empuk para hacker.



4. Migrasi

Secara umum, proses migrasi pada perusahaan berskala SMB dapat berlangsung selama 30-60 hari untuk sistem operasi, dan sekitar 90 hari untuk keseluruhan aplikasi hingga semuanya berjalan normal. Terdapat 4 langkah untuk migrasi dari Windows Server 2003/R2. Untuk mengetahui langkah-langkah migrasi dari Windows Server 2003 ke Windows Server 2012 atau Windows Azure, anda dapat mengunjungi langkah-langkah migrasi dari windows server 2003.
